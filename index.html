Hanh Linux


What is Hanh Linux?

A Linux distribution based on Artix Linux and uses OpenRC as system init. Hanh Linux uses LFS/BLFS guide to build.

What does Hanh Linux have?

Actually, Hanh Linux has: - A command-line interface with an installer - x64 only glibc, gcc and gcc-libs (of course you can install the x86_64 by following our guide)
