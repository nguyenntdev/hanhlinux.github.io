Hanh Linux


What is this?

A Linux distribution based on Artix by using LFS/BLFS guide to build, supports non-multilib x86_64 libraries with rolling-release model and have a guided installer for users.


Purpose

Simplicity: Hanh Linux tries to make it simple without removing a lot of base and network-related packages.
Low-powered devices: Based on Artix (also based on Arch), the system speed is much faster than other popular distribtions. Moreover, Hanh Linux uses x64 gcc-libs and glibc instead of multilib, openrc instead of systemd to make it usable on low-powered devices.
systemd-free: systemd comes as a modern init daemon for Unix-like, but it controls everything than a regular one. Hanh Linux uses openrc, a init does its regular job with more powerful features.
Cuztomization: Hanh Linux only give you basic packages, a text editor and simple tools for network management. Users can decide what they want to install to their system
