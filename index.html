What is this?

A rolling-release Linux distribution based on Artix Linux, using LFS/BLFS guide as a refrence to build, supports non-multilib x86_64 core libraries, with a guided installer.

Purpose

Simplicity: Hanh Linux tries to be minimal without removing a lot of base or network-related packages.

Low-powered devices: Based on Artix Linux, it's much faster than most other popular linux distribtions. Moreover, Hanh Linux uses x64 gcc-libs and glibc instead of multilib, and openrc instead of systemd to make it usable on lower-powered devices.

systemd-free: systemd comes as a modern init daemon for Unix-like, but it does not follow UNIX philosophy, Hanh Linux uses openrc, a init with more powerful features.

Customization: Hanh Linux only give you base packages, a text editor and simple tools for network management, Users can decide what they want to install to their system.
